A wrapper of FRAUDAR algorithm¶

This package implements a wrapper of FRAUDAR algorithm to provide APIs defined in Review Graph Mining project.

Installation¶

Use pip to install this package.

pip install --upgrade rgmining-fraudar

Usage¶

This package provides a review graph class ReviewGraph, and it takes two arguments; a number of blocks and a sub algorithm computing degree of a matrix. See the original article for more information about those arguments.

To construct a review graph instance with \(n\) blocks and aveDegree algorithm,

import fraudar
graph = fraudar.ReviewGraph(n, fraudar.aveDegree)

The constructed graph object has as same interface as other graph classes in Review Graph Mining project.
